# 贪吃蛇游戏

一个使用HTML5 Canvas和JavaScript实现的经典贪吃蛇游戏。

## 特点

- 响应式设计，支持移动设备
- 触摸控制支持
- 键盘控制支持（方向键和WASD）
- 分数系统和最高分记录
- 游戏暂停功能
- 优雅的视觉效果

## 如何运行

1. 克隆仓库：
```bash
git clone [你的仓库URL]
```

2. 在浏览器中打开 `index.html` 文件

## 游戏控制

### 电脑端
- 使用方向键或WASD控制蛇的移动
- 空格键暂停/继续游戏

### 移动端
- 滑动屏幕控制蛇的移动方向
- 点击"开始游戏"按钮开始新游戏
- 点击"游戏规则"按钮查看规则

## 游戏规则

1. 使用方向键或滑动控制蛇的移动
2. 吃到食物得10分
3. 撞墙或撞到自己游戏结束
4. 按空格键暂停/继续游戏
